As an environmentally friendly solution, I wanted to use rechargeable 1.2V NiMH cells instead of the (regular, non-rechargeable) 1.5V batteries. Long story short: I bought a set of Sanyo Eneloop rechargeable batteries. Luckily, they do work great!
